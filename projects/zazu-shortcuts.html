Zazu Shortcuts

About

Zazu Shortcuts is a plugin for the Zazu launcher that allows you to launch programs and URLs using custom aliases. I built it because I wanted a quicker way of navigating around our infrastructure at work and executing commonly used commands. Some examples of shortcuts I use are opening a TortoiseSVN log for a particular code branch or revision, opening the issue details page for a set of Jira issues, opening a solution in an elevated Visual Studio instance and viewing TeamCity build results for a specific branch. It saves myself and others on my team a lot of repetitive typing each day.

Zazu is an Electron application and Zazu Shortcuts is written in JavaScript as a result. Zazu plugin configuration is stored in JSON file. To make it easier to develop the plugin, I defined some TypeScript declaration files (.d.ts) for Zazu's APIs and the configuration file schema. In combination with JSDoc annotations in the plugin source code, this enables Visual Studio Code (my editor of choice) to provide type checking while working in the Zazu Shortcuts codebase. This helps catch mismatched type bugs that dynamically-typed languages such as JavaScript are susceptible to.
