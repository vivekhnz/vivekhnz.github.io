Project ARENA

About

Project ARENA is a top-down shooter game developed using the Unity engine. I built it for the Introduction to Computer Game Design course at University of Technology Sydney in about 3 weeks. During the 2016 UTS games showcase event, Project ARENA was awarded 1st place out of over 15 other games in a class of more than 80 students.
